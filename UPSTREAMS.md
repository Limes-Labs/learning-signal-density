# Upstreams And Reuse Boundary

This repository is original Limes Labs code unless explicitly stated otherwise.
The projects below were inspected for structure, workflow, or scientific
framing. No source code was copied from them into this repository.

| Source | Role | Reuse |
| --- | --- | --- |
| `Limes-Labs/the-broadcast-ceiling` | Serious public research repo shape: tests, paper, result artifacts, sober claims | Structural inspiration only |
| `Limes-Labs/limes-autoresearch` | Ledgered experiment specs, no-cheating language, result-card workflow | Structural inspiration only |
| `metaforismo/benchforge` | Local-first benchmark factory and verifier-ready challenge discipline | Structural inspiration only |
| `karpathy/nanoGPT` | Future small-transformer backend candidate | Not used in current code |

Primary literature used in the first research map is listed in
`docs/literature.md` and `paper/references.bib`.

If future work imports source code, models, datasets, or benchmark items, this
file must be updated before the import lands.
